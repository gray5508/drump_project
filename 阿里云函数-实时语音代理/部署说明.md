# 阿里云函数计算部署说明

部署地域选择“华北 2（北京）”，创建“Web 函数”。

建议配置：

- 运行环境：Node.js 20
- 启动命令：`npm start`
- 监听端口：`9000`
- 内存：256 MB
- 执行超时：120 秒
- 实例并发度：10
- 最小实例数：0
- HTTP 触发器：允许 GET，开启公网地址，测试阶段选择无需认证

上传 `drump-qwen-realtime-fc.zip` 后，在函数配置的环境变量中添加：

- `DASHSCOPE_API_KEY`：千问 API Key
- `ALLOWED_ORIGINS`：`https://gray5508.github.io`
- `MAX_CONNECTIONS`：`10`

函数必须监听 `0.0.0.0:9000`。部署后先访问函数公网地址的 `/health`，返回 `ready: true` 才表示环境变量生效。

不要把 API Key 写进 ZIP、GitHub 或前端 JavaScript。
